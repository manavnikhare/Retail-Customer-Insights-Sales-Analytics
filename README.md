# 🛍️ Customer Behaviour Analysis Report

A complete **Data Analytics** project that analyzes customer shopping behavior using **Python (Pandas), PostgreSQL (SQL), and Power BI**. The project demonstrates an end-to-end analytics workflow, starting from raw data cleaning to business insights and interactive dashboard creation.


---

## 📌 Project Overview

The objective of this project is to understand customer purchasing behavior and identify key patterns that can help businesses improve decision-making.

The project covers:
- Data cleaning and preprocessing using **Python (Pandas)**
- Loading cleaned data into **PostgreSQL**
- Performing business analysis using **SQL**
- Creating an interactive **Power BI Dashboard**
- Generating actionable business insights

---

Businesses collect large amounts of customer transaction data, but raw data alone cannot support decision-making. This project transforms raw customer shopping data into meaningful insights by cleaning the dataset, storing it in PostgreSQL, analyzing it using SQL, and visualizing the results in Power BI.

The project answers important business questions such as:
- Which customer segment generates the highest revenue?
- Do subscribed customers spend more than non-subscribers?
- Which products are most frequently purchased?
- How effective are discounts in driving purchases?
- Which age groups contribute the most revenue?
- Which products receive the highest customer ratings?

The project answers several real-world business questions, including:

- Revenue generated by male vs female customers
